# 🧠 Orientación a Objetos II

Repositorio de prácticas para la materia **Orientación a Objetos II** – Facultad de Informática (UNLP)

---

## 📌 Objetivo

Aplicar conceptos avanzados de desarrollo orientado a objetos mediante la implementación de:

* Patrones de diseño
* Técnicas de refactoring
* Testing y TDD
* Uso de frameworks OO

---

## 📚 Contenidos

* **Diseño OO**: UML, modelado, procesos iterativos
* **Patrones de diseño**: creacionales, estructurales y de comportamiento
* **Refactoring**: code smells y mejoras de diseño
* **Frameworks**: IoC, inyección de dependencias
* **Testing**: unit testing, mocks y TDD

---

## 🛠️ Tecnologías

* Lenguaje: Java
* Testing: JUnit
* IDE: Eclipse

---

## 📂 Estructura
```plaintext
objetos2/

├── refactoring/             ← Ejercicios de refactoring y TDD
